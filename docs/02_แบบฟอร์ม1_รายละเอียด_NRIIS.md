## ส่วนที่ 2 รายละเอียด

**1. ที่มาของแนวคิด**
จังหวัดสิงห์บุรีเป็น "เมืองรอง" ที่มีแหล่งท่องเที่ยวเชิงประวัติศาสตร์ วัฒนธรรม และสายทำบุญที่มีคุณค่า (เช่น วัดพระนอนจักรสีห์ฯ อนุสาวรีย์วีรชนค่ายบางระจัน) แต่ขาดเครื่องมือดิจิทัลที่ช่วยให้นักท่องเที่ยววางแผนเที่ยวได้ง่ายและเหมาะกับตนเอง แพลตฟอร์มท่องเที่ยวที่มีอยู่ส่วนใหญ่ทำหน้าที่เป็น "ไกด์บุ๊กออนไลน์" ให้ผู้ใช้ค้นหาและจัดทริปเอง ทีมจึงคิดค้นและพัฒนา Sing Journey ขึ้นเอง เป็นแพลตฟอร์มที่ใช้ AI เข้าใจผู้ใช้และวางแผนทริปให้อัตโนมัติ พร้อมเพิ่มความสนุกด้วยระบบเกมิฟิเคชัน และยกระดับความแม่นยำของข้อมูลด้วยการดึงเวลาเปิด-ปิดจริงแบบเรียลไทม์

**2. วัตถุประสงค์**
1. สร้างแพลตฟอร์มที่ใช้ AI วิเคราะห์ไลฟ์สไตล์ผู้ใช้และสร้างเส้นทางท่องเที่ยวจังหวัดสิงห์บุรีแบบเฉพาะบุคคล
2. รวมการนำเสนอ–วางแผน–นำทางบนแผนที่จริงไว้ในที่เดียว แก้ไขได้ง่ายด้วยการลากวาง
3. ส่งเสริมการท่องเที่ยวเมืองรองและกระจายรายได้สู่ชุมชน วัด และร้านค้าท้องถิ่นผ่านระบบเกมิฟิเคชัน

**3. สถานะการพัฒนา:** ☑ เป็นสิ่งประดิษฐ์ใหม่ที่คิดค้นขึ้นเอง
ทีมออกแบบและพัฒนาระบบขึ้นใหม่ทั้งหมด ไม่ได้ดัดแปลงหรือต่อยอดจากซอฟต์แวร์ของผู้อื่น (อ้างถึงแพลตฟอร์มท่องเที่ยวชุมชน CBT Thailand เพียงเพื่อเปรียบเทียบให้เห็นความต่างเชิงแนวคิดเท่านั้น) จุดที่เป็นนวัตกรรมใหม่ ได้แก่ (1) AI เข้าใจผู้ใช้รายบุคคล (2) วางแผนแบบลากวาง (3) วาด Road Map บนแผนที่จริง (4) เช็คอินรับไอเทม 3 มิติ (5) หมวดท่องเที่ยวสายทำบุญ Watt's Up! (6) ระบบดึงข้อมูลเปิด-ปิดจริงแบบเรียลไทม์ ผสานข้อมูลอัตโนมัติ + การแจ้งจากคนในท้องถิ่นผ่าน LINE OA

**4. คุณสมบัติ/คุณลักษณะเฉพาะและขอบเขตการใช้งาน**
- เว็บแอปพลิเคชัน (PWA) ใช้งานบนมือถือและคอมพิวเตอร์ผ่านเบราว์เซอร์ ไม่ต้องติดตั้งแอป
- เข้าสู่ระบบด้วย **LINE Login** (ผูกบัญชี LINE ของผู้ใช้)
- **แจ้งเตือนสถานะเปิด-ปิดผ่าน LINE** — ผู้ใช้รับการแจ้งเตือน และเจ้าของร้าน/ตลาด/คนในท้องถิ่นแจ้งสถานะเข้ามาได้ผ่าน LINE Official Account
- ปัจจุบันมีฐานข้อมูลจริง **12 สถานที่ ครอบคลุมทั้ง 6 อำเภอ** ของจังหวัดสิงห์บุรี, **6 เทศกาล/งานประจำจังหวัด** และข้อมูลวัด **6 แห่ง** ในหมวด Watt's Up!
- รองรับภาษาไทย (และต่อยอดภาษาอังกฤษได้)

**5. จุดเด่น/กลไกที่แตกต่าง**
- **AI Personalization:** เข้าใจว่าผู้ใช้เป็นใคร สไตล์ชีวิตแบบไหน แล้วเสนอแผนหลายรูปแบบ
- **Map-based Roadmap:** วาดเส้นทางจริงบนแผนที่ จากบ้าน → จังหวัด → แตกเป็นรายอำเภอ → ไล่จุดต่อเนื่องได้เกิน 1 วัน พร้อมแสดงโรงแรมใกล้จุดเที่ยว
- **ปรับแก้ลากวาง:** เพิ่ม/ลบ/สลับลำดับสถานที่ได้ทันที
- **ข้อมูลจริงแบบเรียลไทม์ (Real-time Info Sync) 2 ชั้น:** (ก) ดึงเวลาเปิด-ปิดและสถานะเปิด-ปิดชั่วคราวอัตโนมัติจาก Google Places API (ข) **รับแจ้งสถานะจากคนหน้างานจริงผ่าน LINE OA** (เจ้าของร้าน/ตลาด/ชุมชน เช่น ปิดกะทันหัน ติดงานวัด น้ำท่วม) ระบบแสดงป้าย "เปิดอยู่/ใกล้ปิด/ปิดแล้ว" และ AI หลีกเลี่ยงการจัดสถานที่ที่ปิด เพิ่มความแม่นยำและกัน "ไปถึงแล้วปิด"
- **แจ้งเตือนผ่าน LINE:** ส่งแจ้งเตือนการเปลี่ยนสถานะของสถานที่ในแผนของผู้ใช้ผ่าน LINE Messaging API
- **เกมิฟิเคชัน:** เช็คอินรับไอเทม 3 มิติ (ของที่ระลึก / คูปองส่วนลดแบบใช้แล้วหมดไป)
- **Watt's Up!:** หมวดสายทำบุญ พร้อมประวัติวัดและพระเกจิดัง

**6. หลักการ วิธีการ และขั้นตอนการทำงาน**
1. ผู้ใช้เข้าสู่ระบบด้วย LINE Login และตอบคำถามสั้นๆ / ระบบเรียนรู้จากพฤติกรรม
2. ส่งโปรไฟล์ผู้ใช้ + ฐานข้อมูลสถานที่ ให้โมเดล AI ประมวลผล
3. ระบบดึงข้อมูลจริงแบบเรียลไทม์ (เวลาเปิด-ปิด/สถานะ) จาก Google Places API ผสานกับสถานะที่คนในท้องถิ่นแจ้งผ่าน LINE OA เพื่อกรองเฉพาะสถานที่ที่เปิดให้บริการในช่วงเวลานั้น
4. AI คืนผลเป็นชุดเส้นทาง (roadmap) หลายแบบในรูปแบบข้อมูลที่มีโครงสร้าง
5. ระบบวาดเส้นทางบนแผนที่ พร้อมแนะนำโรงแรม/ร้านอาหารใกล้เคียง และแสดงป้ายเวลาเปิด-ปิด
6. ผู้ใช้ปรับแก้แผนด้วยการลากวาง และรับแจ้งเตือนผ่าน LINE เมื่อสถานะสถานที่ในแผนเปลี่ยน
7. เมื่อเดินทางไปถึงและเช็คอิน ระบบมอบไอเทม 3 มิติเก็บใน collection

**7. วัสดุและอุปกรณ์ที่ใช้** (เทคโนโลยี)
เว็บเฟรมเวิร์ก Next.js/React, แผนที่ MapLibre GL JS, โมเดล 3 มิติ model-viewer, โมเดลภาษา Claude API, ข้อมูลเวลาเปิด-ปิดจริง Google Places API, การยืนยันตัวตน/แจ้งเตือน **LINE Login + LINE Messaging API (LINE OA)**, ฐานข้อมูล/จัดเก็บ Supabase, โฮสต์บน Vercel — เครื่องมือส่วนใหญ่เป็นโอเพนซอร์ส/มีแพ็กฟรี ต้นทุนต่ำ

**8. ประโยชน์และคุณค่า**
ลดเวลาและความยุ่งยากในการวางแผนทริป เพิ่มความแม่นยำในการเดินทางด้วยข้อมูลเปิด-ปิดจริง 2 ชั้น (อัตโนมัติ + คนในพื้นที่) กระตุ้นการท่องเที่ยวเมืองรอง กระจายรายได้สู่ชุมชน/วัด/ร้านค้า ส่งเสริมการเรียนรู้ประวัติศาสตร์และวัฒนธรรมท้องถิ่น และเปิดช่องให้ชุมชนมีส่วนร่วมดูแลข้อมูลของตนเองผ่าน LINE
